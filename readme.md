# SpectrumWaterfallMonitor

WPF-застосунок для візуалізації спектра радіосигналів у реальному часі
з підтримкою класичного спектра та водоспада, який відображає історичні дані

Проєкт має наступну структуру:
- Core - бізнес-логіка, генерація та моделі
- Controls - WPF-контроли
- Wpf - UI, MVVM, моки для відображення

## Структура проєкту

```
├── SpectrumWaterfallMonitor.Core
├── SpectrumWaterfallMonitor.Controls
├── SpectrumWaterfallMonitor.Wpf

```
